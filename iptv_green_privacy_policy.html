Politique de confidentialité

Dernière mise à jour : [À REMPLIR]

Nous vous remercions d’utiliser IPTV GREEN. Cette politique explique quelles données nous pouvons collecter, comment nous les utilisons et les options dont vous disposez. En utilisant l’application, vous acceptez les pratiques décrites ci‑dessous.

1. Données collectées

IPTV GREEN peut collecter les éléments suivants :

  • Données techniques : modèle d'appareil, version Android, informations de diagnostic (crash, performance).
  • Données d’utilisation anonymisées : ouvertures d’application, actions générales pour améliorer l’expérience.

IPTV GREEN ne collecte pas de nom, d'email, d'adresse ou de données sensibles, sauf si l'utilisateur les saisit volontairement (par exemple via un formulaire de contact).

2. Utilisation des données

Ces données servent uniquement à :

  • assurer le fonctionnement et la stabilité de l’application ;
  • analyser les performances et corriger des problèmes techniques ;
  • ne pas personnaliser l’utilisateur à des fins commerciales.

3. Partage et tiers

Nous ne vendons ni ne partageons vos données personnelles. Nous pouvons toutefois recourir à des prestataires techniques (ex. Google Play Services, Crashlytics) qui traitent des données d’usage anonymisées pour l'analyse et la sécurité.

4. Permissions

L'application peut demander :

  • Accès Internet : pour lire les flux IPTV fournis par l'utilisateur ou disponibles publiquement.
  • Accès au stockage (optionnel) : si l'utilisateur importe une playlist (fichier .m3u).

5. Stockage et sécurité

Toutes les données collectées sont stockées localement sur l'appareil ou traitées anonymement par des services tiers. Nous appliquons des mesures techniques raisonnables pour protéger ces données contre l'accès non autorisé.

6. Contenu et responsabilité

IPTV GREEN ne fournit aucun contenu vidéo. L'utilisateur est responsable des playlists qu'il importe. L'application est conçue pour fonctionner avec des flux libres de droits ou autorisés par les détenteurs de contenu.

7. Droits de l'utilisateur

Vous pouvez :

  • désinstaller l'application pour arrêter toute collecte ;
  • supprimer toute playlist importée depuis l'interface de l'app.

8. Services tiers

L'application utilise potentiellement les services suivants : Google Play Services, Firebase (si activé). Ces services disposent de leurs propres politiques de confidentialité.

Pour toute question relative à cette politique, contactez le développeur : [EMAIL_DÉVELOPPEUR].

Instructions rapides pour publier cette page (GitHub Pages)

  1. Créez un dépôt GitHub (ex. iptv-green-privacy).
  2. Ajoutez ce fichier en le nommant index.html à la racine du dépôt.
  3. Dans les paramètres du dépôt → Pages, sélectionnez la branche main (ou gh-pages) et validez. GitHub Pages publiera la page sous https://votre-nom-utilisateur.github.io/nom-depot/.
  4. Copiez l'URL générée et collez-la dans la section "Politique de confidentialité" de Google Play Console.

Si vous préférez, vous pouvez aussi utiliser Google Sites : créez un site, collez le HTML dans une page via l'éditeur ou importez le PDF, publiez et récupérez l'URL publique.
